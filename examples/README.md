# Примеры сценариев Scenaria

Готовые `.feature` для первого знакомства с программой. Все примеры используют публичный сайт [example.com](https://example.com) — не требуют логина и работают из коробки.

## Как открыть в Scenaria

1. Запустите приложение → вкладка **«Старт»**
2. Нажмите **«Открыть примеры сценариев»**
3. В дереве слева выберите файл и нажмите **Запустить тест** (Ctrl+Enter)

Или вручную: **Файл → Открыть проект…** и укажите эту папку `examples`.

## Файлы

| Файл | Что показывает |
|------|----------------|
| `01-pervaya-proverka.feature` | Открыть страницу, проверить заголовок, закрыть браузер |
| `02-perehod-po-ssylke.feature` | Клик по ссылке, пауза, «назад», проверка |
| `03-uslovnyy-shag.feature` | Блок **«Если вижу …»** с вложенными шагами |
| `04-tablica-primerov.feature` | **Таблица примеров**: «Структура сценария» + `Примеры:` (2 прогона) |

## Таблица данных

В `04-tablica-primerov.feature` один шаблон шагов и таблица `Примеры:` — Scenaria выполнит сценарий **для каждой строки** (в каталоге бейдж «2 прим.»). Плейсхолдеры `<url>` подставляются из колонки таблицы.

Подробнее: **Справка → Справка…** (F1) → категория **«Данные и таблицы»**.

## Советы новичкам

- Перед запуском замените URL в примере на свой сайт, когда будете готовы тестировать реальный проект.
- Полный список шагов: **Ctrl+Space** в редакторе или **Справка → Справка…** (F1), в том числе категория **«Данные и таблицы»**.
- Запись с нуля: **Открыть браузер** → **Начать запись** → действия на сайте → **Остановить запись**.
